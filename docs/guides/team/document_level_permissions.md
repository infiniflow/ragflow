# Document-level permissions

Configure document-level permissions only in RAGFlow Enterprise.

---

Document-level permissions control access and operations at the document level. They provide finer-grained control than resource-level sharing.

## Configure operations

Configure which operations users can perform on documents.

## Document permission levels

Document permission levels define how users can access and operate on documents.
