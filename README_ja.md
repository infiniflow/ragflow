## 💡 MetaGrossAIとは？
**MetaGrossAI** は、最先端のRAG（Retrieval-Augmented Generation）とエージェント機能を融合させ、LLMのための優れたコンテキストレイヤーを構築する最先端のRAGエンジンです。あらゆる規模の企業に適応可能な、合理化されたRAGワークフローを提供します。統合されたコンテキストエンジンと構築済みのエージェントテンプレートにより、開発者は複雑なデータを高精度な本番環境レベルのAIシステムへと効率的に変換できます。

## 🌟 主な機能
### 🍭 **「質の高い入力から、質の高い出力へ」**
- 複雑なフォーマットの非構造化データから、深いドキュメント理解に基づいた知識抽出を行います。
- 無制限のトークンから「データという干し草の山から針を探す」ことができます。

### 🍱 **テンプレートベースのチャンキング**
- インテリジェントで説明可能です。
- 豊富なテンプレートオプションから選択できます。

### 🌱 **ハルシネーションを削減する根拠のある引用**
- テキストチャンキングの可視化による人間の介入の許可。
- 根拠のある回答を裏付けるための主要な参考文献と追跡可能な引用のクイックビュー。

### 🍔 **多様なデータソースとの互換性**
- Word、スライド、Excel、テキスト、画像、スキャンデータ、構造化データ、Webページなどをサポートします。

### 🛀 **自動化された効率的なRAGワークフロー**
- 個人および大企業向けにカスタマイズされた合理的なRAGオーケストレーション。
- 構成可能なLLMおよび埋め込みモデル。
- 融合された再ランク付けと組み合わせた複数回のリコール。

- RAM >= 16 GB
- Disk >= 50 GB
- Docker >= 24.0.0 & Docker Compose >= v2.26.1
