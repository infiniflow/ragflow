## Install front-end dependencies

   ```bash
   npm install
   ```

## Launch front-end

   ```bash
   npm run dev
   ```

   _The following output confirms a successful launch of the system:_

   ![](https://github.com/user-attachments/assets/0daf462c-a24d-4496-a66f-92533534e187)

## Shutdown front-end

   Ctrl + C or

   ```bash
   kill -f "umi dev"
   ```
